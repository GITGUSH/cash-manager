# Cash Manager

Personal cash management system developed in C# with ASP.NET and PostgreSQL. 

## Technologies 
- C# / ASP.NET Minimal API
- PostgreSQL
- Npgsql
- BCrypt.Net

## How to run local

1. Clone the repository
2. Create the PostgreSQL database
3. Execute the SQL scripts in /scripts folder
4. Setup the .env file with the database credencials.
5. Run the project with 'dotnet run'

## Modules
- Usuário (User)
- Conta (Account)
- Categoria (Category)
- Operação (Operation)
